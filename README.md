# IVCAREMEDIKA — Official Website

A premium, modern healthcare landing page built with React and Next.js, serving as the official digital presence for IVCareMedika.

---

## 🩺 Description

This is the official landing page and healthcare portal for **IVCareMedika**, a professional homecare provider. The platform showcases premium homecare services, including:
*   **Homecare Infusion Services** (Layanan Infus Homecare)
*   **Vitamin Therapy** (Terapi Vitamin & Boosters)
*   **Medical Wellness Services** (Kesehatan & Kebugaran Medis)
*   **Healthcare Consultation** (Konsultasi Medis Profesional)

---

## 🎯 Website Goals

*   **Improve Brand Trust**: Establish a highly credible online reputation matching international clinical standards.
*   **Professional Healthcare Presence**: Display transparent service information, clinical details, and pricing.
*   **Improve Online Visibility**: Utilize modern semantic HTML and SEO strategies to reach patients effectively.
*   **Increase WhatsApp Booking Conversion**: Streamline the booking flow with prominent, intuitive, and responsive CTA elements.

---

## 💻 Tech Stack

*   **Framework**: Next.js 16 (App Router + Turbopack)
*   **Library**: React 19
*   **Styling**: Tailwind CSS v4
*   **Language**: TypeScript
*   **Icons & Animation**: Lucide React & Framer Motion

---

## ✨ Features

*   📱 **Mobile-First Layout**: Fully responsive experience optimized for all device sizes (phones, tablets, desktops).
*   💎 **Premium Healthcare UI**: Elegant, airy design using harmonious white and soft blue medical palettes.
*   ⚡ **WhatsApp Call-To-Action (CTA)**: Immediate and easy booking integrations via customized WhatsApp links.
*   🌀 **Smooth Micro-Animations**: Interactive hover effects and scroll transitions powered by Framer Motion.
*   🔍 **SEO Optimized**: Properly structured header tags, meta-descriptions, and clean, fast-loading architecture.
*   💬 **Interactive FAQ & Booking**: Accordion-based FAQs and step-by-step homecare booking flow.

---

## 🎨 Design Direction

*   **Airy Medical Aesthetics**: Inspired by top-tier global medical wellness brands, using crisp whites, deep navies, and soft healing blues.
*   **Modern Startup Cleanliness**: Sleek typography, generous whitespace, and high-quality iconography to evoke comfort and professionalism.
*   **Intuitive Hierarchy**: Focus on readability, simple navigation, and trust indicators (clinical certifications, FAQs, clear testimonials).

## 🛠️ Development Setup

First, install the clean and verified package dependencies:
```bash
npm install
```

To run the development server with Hot Module Replacement (Turbopack):
```bash
npm run dev
```

Open [http://localhost:3001](http://localhost:3001) in your browser to view the application.

---

## 📦 Build & Production

To generate a highly-optimized, static production bundle:
```bash
npm run build
```

To start the production server locally:
```bash
npm run start
```
